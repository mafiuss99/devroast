# DevRoast

Brutally honest code reviews.

## Features

- **Paste & Roast**: Submit your code and get an honest score (0-10)
- **Roast Mode**: Maximum sarcasm enabled with detailed critiques
- **Shame Leaderboard**: See the worst code on the internet ranked by shame
- **Analysis Cards**: Detailed feedback on code issues with suggested fixes

## Tech Stack

- Next.js 16 (App Router)
- Tailwind CSS v4
- Biome (lint/format)
- Base UI (headless components)
- Shiki (syntax highlighting)
- React Circular Progressbar

## Getting Started

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)
